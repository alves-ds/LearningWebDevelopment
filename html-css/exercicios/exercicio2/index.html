Parágrafos e quebras de linha

Para criar novos parágrafos adotamos a tag < p >

Nós também podemos quebrar um texto utilizando a tag <br>

Um exemplo disso é este texto. Vamos continuar na próxima linha:
desta maneira.

Símbolos especiais

Podemos também utilizar símbolos especiais utilizando & antes do seu chamado, da seguinte forma:
®
Δ
©
™
¥
↑
€

Emojis

Podemos também adicionar emojis, buscando o código deles no site emojipedia e colocando após &#x, um exemplo é:
❤ 🔥 ✅ ✨
