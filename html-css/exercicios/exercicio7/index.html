Trabalhando com listas

Listas ordenadas

Podemos criar listas ordenadas utilizando a tag ol, da seguinte forma:

  1. Acordar
  2. Lavar o rosto
  3. Tomar café
  4. Escovar os dentes
  5. Ligar o computador

Listas não ordenadas

Mas podemos também criar listas não ordenadas utilizando a tag ul, da seguinte forma:

  • Pão
  • Tomate
  • Cenoura
  • Café
