Listas dentro de listas

Podemos colocar listas dentro de listas, da seguinte forma:

    • Abacate
    • Abacaxi
    • Mamão
    • Limão

Links externos

Podemos também colocar links externos no nosso site, utilizando-se da tag a e passando o parâmetro href, da seguinte forma:

Você pode acessar o meu repositório público no Github

Mas, se não passarmos os parâmetros target e rel, quando clicarmos no link ele fechará o nosso site, portanto, vamos adicionar estas tags:

Você pode acessar o meu repositório público no Github sem sair desse site, clicando aqui

Links internos (navegar entre páginas de um mesmo site)

Também podemos referenciar páginas que pertençam ao nosso site, para permitir uma "navegação interna", da seguinte forma:

Clique aqui e vá para a página 2 do site!

Você também pode acessar a nossa página de notícias, clicando aqui!
